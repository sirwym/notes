```cpp
#include <bits/stdc++.h>
using namespace std;

int divide(int a, int b) {
    if (b == 0) throw runtime_error("除零错误");
    return a / b;
}

int main() {
    // 基本 try-catch
    try {
        int result = divide(10, 0);
        cout << result << endl;
    } catch (const runtime_error &e) {
        cerr << "捕获异常: " << e.what() << endl;
    }

    // 多种异常类型
    try {
        int arr[5] = {1,2,3,4,5};
        int idx;
        cin >> idx;
        if (idx < 0 || idx >= 5) throw out_of_range("数组越界");
        cout << arr[idx] << endl;
    } catch (const out_of_range &e) {
        cerr << e.what() << endl;
    } catch (...) {
        cerr << "未知异常" << endl;
    }

    return 0;
}
```

## 📖 要点说明

- `try` 包裹可能出错的代码，`catch` 捕获并处理
- `throw` 抛出异常，`catch` 按类型匹配
- `catch(...)` 捕获所有未处理的异常
- 竞赛中几乎不用异常处理，但 C++ 语法要了解

### ⚠️ 常见错误

- catch 类型不匹配导致异常未被捕获
- throw 在 try 外抛出，程序直接终止
- 竞赛场景下异常处理影响性能，不建议常用
