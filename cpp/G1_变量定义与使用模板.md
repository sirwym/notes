```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int a;           // 未初始化，值不确定！
    int b = 10;      // 声明并初始化
    int x = 1, y = 2, z = 3;  // 多变量同时声明

    const int MAXN = 1000;   // 常量，不可修改

    int score = 0;
    score += 10;    // 等价于 score = score + 10
    score++;        // 加 1

    // 变量交换
    int m = 3, n = 5;
    swap(m, n);     // m=5, n=3

    cout << b << " " << score << endl;
    return 0;
}
```

## 📖 要点说明

- 变量必须先声明后使用，声明时可以不赋值（但值不确定）
- 良好命名：`score`、`maxNum`，不用 `a1`、`x2`
- `const` 定义的常量不可修改，适合定义数组大小等
- `swap(a, b)` 是 C++ 内置交换函数

### ⚠️ 常见错误

- 变量未初始化就使用，得到垃圾值
- 变量名与关键字冲突如 `int int = 5;`
- 忘加分号 `;` 是最常见的编译错误
