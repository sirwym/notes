```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int a = 5, b = 3;
    cout << (a > b) << endl;    // 1 (true)
    cout << (a < b) << endl;    // 0 (false)
    cout << (a >= b) << endl;   // 1
    cout << (a == b) << endl;   // 0
    cout << (a != b) << endl;   // 1

    // 连续比较要拆开
    int x = 5;
    if (x > 1 && x < 10) {
        cout << "x 在 1~10 之间" << endl;
    }

    // 浮点数比较用极小值
    double d1 = 0.1 + 0.2, d2 = 0.3;
    if (fabs(d1 - d2) < 1e-9) {
        cout << "近似相等" << endl;
    }

    return 0;
}
```

## 📖 要点说明

- 关系运算结果是 `bool`：`true`(1) 或 `false`(0)
- 连续比较 `1 < x < 10` 不是数学含义，要拆成 `x > 1 && x < 10`
- 浮点数不要用 `==` 比较，用 `fabs(a-b) < 1e-9`

### ⚠️ 常见错误

- `==` 写成 `=` 变赋值：`if (a = 5)` 永远为真
- 连续比较写法错误，不是数学区间判断
- 浮点数 `==` 可能因精度不相等
